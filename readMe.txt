Preconditions:

1. Please update following variable values in Git_Project.py file:

username = "xxxxx"
password = "yyyyy"

Replace xxxxx to your github username and yyyyy to your github password.

2. Run Git_Project.py in python3. (Selenium library should install installed in python)

3. Git_Project.py file use chrome browser to automate "https://github.com/login"
